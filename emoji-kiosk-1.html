Step 4 — Lock the Phone (Kiosk Mode)
Option A — Screen Pinning (easiest, built-in):
Open the app in Chrome → go to Settings → Security → Screen Pinning → enable it. Then open Recent Apps, find Chrome, and tap the pin icon. The phone is now locked to this page. To unpin: hold Back + Recents simultaneously.

Option B — Guided Access style via Chrome:
Add this page to the home screen (Chrome menu → "Add to Home Screen"). Then enable Screen Pinning as above. The app will launch full-screen with no browser chrome.

Tip: Set the Chrome homepage to this file's URL so it always opens here after a reboot. You can host the file for free on GitHub Pages — just put it in a repo and enable Pages. Each child gets their own file with their own token and name.
