Step 4 — Lock the Phone (Kiosk Mode)
Option A — Screen Pinning (easiest, built-in):
Open the app in Chrome → go to Settings → Security → Screen Pinning → enable it. Then open Recent Apps, find Chrome, and tap the pin icon.

Option B — Add to Home Screen:
Add this page to the home screen from Chrome, then use Screen Pinning so the app opens fullscreen.
